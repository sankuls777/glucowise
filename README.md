Type 1 Diabetes Insulin Dosage Support App - 
GlucoWise is a decision-support tool only. It does not prescribe insulin. All dose
estimates must be reviewed against doctor-approved personal settings before acting.
This specification is for technical and clinical review — not for patient self-use without
endocrinologist guidance.

GlucoWise is a mobile-first, AI-assisted insulin dosage support application designed for adults with Type 1 
Diabetes who self-manage their insulin therapy. The app helps the user estimate pre-meal 
NovoRapid (insulin aspart) doses, track post-meal glucose responses, identify patterns, and 
share structured reports with their endocrinologist. It does not autonomously prescribe insulin 
or override physician-approved settings.
